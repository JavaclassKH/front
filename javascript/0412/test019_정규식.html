자바스크립트 정규식(Regular Expression) 연습


      형식 : /~~~~/플러그인 , /[a-z]/플러그인 & 자바 : ^[0-9]*$
      플러그인 :  ※기본값은 쓰지 않아도 들어가있음
      g(global) : 전체범위에서의 처리(기본값)
      m(multi line) : 여러줄에 걸쳐서 처리(textarea 태그) + m을 추가로 작성해줌
      s(single line) : 한줄 처리(기본값)
      i : 영문자인경우 대/소문자를 구별하지 않음
      u(unicode) : 유니코드에 대해 구별처리시 사용

      정규식을 실행하기 위한 함수 + 자바에선 'pattern.matches' 사용
      1. 사용자입력 문자열.match(정규표현식 or 표현식담은 변수) : 포함유무에 대한 비교시 사용하면 편리
      2. 정규표현식 변수.test(사용자입력문자열) : 구체적인 문자(열)간의 비교체크시 사용하면 편리

      /* 
      자바스크립트 : /[]/+플러그인  ;
      [^] => 안의 ^는 부정의 의미
      ^[] => 앞의 ^는 '처음(시작)'의 의미
      []+ => 뒤의 +는 1개 이상, *는 0개 이상, ?는 0개이거나 1개.
      []+$ => $는 마지막을 의미. / []뒤의 +$는 []안의 문자가 1개 이상으로 끝나야 한다
      () => 그룹 적용시 괄호 사용
      {} => 범위(시작 크기,끝 크기) 적용 시, {}앞에 쓰이는 '.'은 앞의 내용이 {} 범위 개수만큼 진행하라는 구분자
      ?=.* => 앞쪽의 모두(.)일치를 의미, ()안의 내용이 1개 이상 반복(*) 한다
      w =>  '알파벳(대,소문자)' , '숫자' , '_' 을 의미
      
      
      ※ 자바스크립트에서 한글은 유니코드에 완성된 코드로 등록되어있기에 '특수코드'로 취급하지 않음

      let rMid = /\w.{4,20}/g;   // 아이디는 영문(대,소문자), 숫자, 밑줄만 포함하여 4자에서 20자까지 처리
      let rPwd = /(!@#$%^&*-=[]{};':"<>,.')/g; // 비밀번호는 영문자,숫자 ,특수문자가 최소 1자 이상은 포함되면서 4~20자까지 처리
      let rPwd = /(!@#$%^&*-=[]{};':"<>,.')/g; // 비밀번호는 영문자,숫자 ,특수문자가 최소 1자 이상은 포함되면서 4~20자까지 처리
      let rName = /(!@#$%^&*-=[]{};':"<>,.')/g; // 성명은 영문자와 한글만 4~10자까지 처리

      let reHomepage2 = /^[http|https][:]{1}\[\/]{1,2}[w]{3}\.[a-zA-Z0-9]{1,15}\.[a-zA-Z]{3,5}$/g;
